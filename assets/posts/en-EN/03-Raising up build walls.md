# Raising up walls

Once we know the height of the building we are making, all that's left is to "raise" the walls to start constructing the windows, doors, and all the elements that make up the exterior facades of the buildings.
To do this, knowing the height of the building (seen in the previous section: Calculating building heights), follow these steps:

`VIDEO WEBP 1`

1. In Minecraft, with the wooden axe in hand (use `//wand` if you don't have the axe or grab it from your creative inventory), type the command `//sel poly` to make a polygonal selection.
2. Select the outline you created earlier. To do this, you must start by left-clicking on the first point. Next, add points to the polygon using right-click until you get the desired shape.
3. Make sure the selection is only 1 block high. If there are other constructions around, be careful not to include them in the selection, as they could break. If you cannot select everything at once, you can do it in different steps.
4. Type the command `//stack [height] [direction]` . In **[height]** , you must type the number of blocks left to reach the total height of the building. In **[direction]** , you must type "up" so that the selection is copied upwards.
5. Check that it has been copied correctly and the building height is correct. In case there was any error, you can type the command `//undo` to undo it and try again.
